# Horror Sound Effects

This directory contains ambient horror sound effects for the Burger Bhau website.

## Required Sound Files

Please add the following sound files to this directory:

### ambient-horror.mp3
- Low, unsettling ambient background noise
- Should be subtle and loopable
- Duration: 30-60 seconds
- Volume: Low, atmospheric

### whisper.mp3
- Faint, indistinct whispering sounds
- Short duration (2-3 seconds)
- Used for hover effects on interactive elements

### door-creak.mp3
- Classic horror door creak sound
- Short duration (1-2 seconds)
- Used for navigation clicks

### heartbeat.mp3
- Slow, steady heartbeat sound
- Medium duration (5-10 seconds)
- Can be used for dramatic moments

### static.mp3
- TV/radio static noise
- Short duration (1-3 seconds)
- Used for glitch effects

### glitch.mp3
- Digital glitch/error sound
- Very short duration (0.5-1 seconds)
- Used for text distortion effects

## Usage

These sounds are managed by the `useHorrorSound` hook and `HorrorSoundManager` component. Users can toggle sound on/off using the SoundToggle button in the top-right corner.

## Licensing

Ensure all sound files are properly licensed for web use. Consider using:
- Creative Commons licensed sounds
- Original compositions
- Purchased sound effects

## File Format

- Format: MP3
- Sample rate: 44.1kHz
- Bitrate: 128kbps or higher
- Mono or stereo (stereo preferred for ambient sounds)
